
# 2:1 Political Strategy

## 개요
본 프로젝트는 하버드 정치 전략 이론을 기반으로 한 '2:1 페르소나 분할 전략'을 정리하고, 전 세계 실제 사례를 통해 이를 분석합니다.

## 주요 전략
- **2:1 페르소나 전략**: 한 명의 주요 타겟에 대해 두 명의 역할 분담된 공격 페르소나가 존재
- **공식 연합 없이 전략적 연대 구성**
- **페르소나를 통한 이미지 분리 및 프레임 이식**
